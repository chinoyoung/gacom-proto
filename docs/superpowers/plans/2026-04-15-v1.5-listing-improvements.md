## New Schema Fields

### programs table
- `startingPrice: v.optional(v.number())` — USD number (e.g., 10000)
- `isFree: v.optional(v.boolean())` — displays as "No Upfront Fees" on frontend
- `costVariations: v.optional(v.array(v.object({ label: v.string(), price: v.number() })))` — e.g., [{label: "2 weeks", price: 3000}]
- `paymentTerms: v.optional(v.array(v.string()))` — e.g., ["Payment Plan", "Scholarships"]
- `refundPolicy: v.optional(v.string())` — category name
- `refundPolicyUrl: v.optional(v.string())` — link to external policy
- `depositFee: v.optional(v.number())` — deposit/application fee amount
- `exclusions: v.optional(v.array(v.string()))` — mirrors whatsIncluded
- `optionalInclusions: v.optional(v.array(v.string()))` — which items in whatsIncluded are optional
- `programTags: v.optional(v.array(v.string()))` — e.g., ["LGBTQI+", "Sustainability"]
- `yearFounded: v.optional(v.number())` — provider's founding year

### reviews table
- `pros: v.optional(v.array(v.string()))` — review pros
- `cons: v.optional(v.array(v.string()))` — review cons
